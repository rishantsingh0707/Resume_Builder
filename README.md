# Resume Builder Pro

A modern resume builder built using:

- React
- Vite
- Tailwind CSS
- Node.js
- Express
- PDFKit

## Features

- ATS Score Checker
- Live Resume Preview
- PDF Export
- Local Storage Save
- Responsive Design
